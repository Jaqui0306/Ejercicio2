## 📝 Reflexión
En esta práctica aprendí a usar Flexbox para acomodar imágenes y texto correctamente en la página. También entendí cómo centrar los elementos y ajustar tamaños de letra para que el diseño se viera igual al ejemplo.Tuve problemas con el tamaño de los títulos y la alineación, pero los solucioné cambiando márgenes, tamaños de fuente y eliminando estilos por defecto del navegador.
Al final comprendí mejor cómo pequeños cambios en CSS afectan el diseño completo.
